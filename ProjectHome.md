# PSDG Synthetic Data Generator (PSDG) #

Is a tool which facilitates the generation of random data in a very flexible and powerful way given a target database.

PSDG is based in _probabilistic algorithms, substitution, NULL'ing out, Gibberish, Functions_ and _Shuffling techniques_ to populate the target database given an set of constraints.

---

**Requeriments**

MySQL database (version 5.1.16+) with InnoDB engine enabled. (At moment, PSDG work only with MySQL databases).

Apache (version 2.2+).

PHP (version 5).
